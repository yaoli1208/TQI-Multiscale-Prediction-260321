**设计目标**: 将多尺度分解、深度学习、大模型三层融合，形成端到端可训练的统一框架

## 二、详细设计

### 2.1 Layer 1: 多尺度分解层

**输入**: 原始TQI序列 `X = [x_1, x_2, ..., x_T]`

**分解方法**:
```python
# 方法1: STL分解 (Seasonal-Trend decomposition using Loess)
trend, seasonal, residual = STL_decomposition(X, period=26)  # 26个检测点≈1年

# 方法2: 小波分解 (Wavelet Decomposition)
# 适合捕捉多频率成分

# 方法3: EMD (Empirical Mode Decomposition)
# 适合非线性和非平稳信号
```

**输出**:
- `X_trend`: 年际劣化趋势
- `X_seasonal`: 季节性波动
- `X_residual`: 事件驱动残差（维修、极端天气）

### 2.2 Layer 2: 深度学习特征层

#### 分支1: 趋势分支 (LSTM + Attention)

```python
class TrendBranch(nn.Module):
    """处理年际劣化趋势"""
    def __init__(self, input_dim, hidden_dim, num_layers):
        self.lstm = nn.LSTM(input_dim, hidden_dim, num_layers, 
                           batch_first=True, bidirectional=True)
        self.attention = MultiHeadAttention(hidden_dim * 2)
        self.fc = nn.Linear(hidden_dim * 2, output_dim)
    
    def forward(self, x_trend):
        # x_trend: [batch, seq_len, features]
        lstm_out, _ = self.lstm(x_trend)
        attn_out = self.attention(lstm_out)
        return self.fc(attn_out)
```

**特点**:
- 双向LSTM捕捉前后向依赖
- 自注意力机制识别关键时间点
- 适合长期趋势建模

#### 分支2: 季节分支 (Transformer)

```python
class SeasonalBranch(nn.Module):
    """处理季节性模式"""
    def __init__(self, d_model, nhead, num_layers):
        self.pos_encoding = PositionalEncoding(d_model)
        self.transformer = TransformerEncoder(
            TransformerEncoderLayer(d_model, nhead), 
            num_layers
        )
    
    def forward(self, x_seasonal):
        x = self.pos_encoding(x_seasonal)
        return self.transformer(x)
```

**特点**:
- 位置编码捕捉周期信息
- 多头注意力学习季节间关联
- 适合周期性模式建模

#### 分支3: 残差分支 (1D-CNN + Gating)

```python
class ResidualBranch(nn.Module):
    """处理突发事件残差"""
    def __init__(self):
        self.conv_layers = nn.Sequential(
            nn.Conv1d(in_channels, 64, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.Conv1d(64, 128, kernel_size=3, padding=1),
            nn.ReLU(),
        )
        self.gate = GatedMechanism(128)
    
    def forward(self, x_residual):
        conv_out = self.conv_layers(x_residual)
        return self.gate(conv_out)
```

**特点**:
- CNN捕捉局部突变特征
- 门控机制过滤噪声
- 适合维修事件、异常检测

### 2.3 Layer 3: 大模型融合层

#### 方案A: Lag-Llama 作为顶层融合器

```python
class LagLlamaFusion(nn.Module):
    """使用Lag-Llama进行多尺度特征融合"""
    def __init__(self):
        # 加载预训练Lag-Llama
        self.lag_llama = load_pretrained_lag_llama()
        
        # 投影层：将多尺度特征映射到Lag-Llama输入空间
        self.projection = nn.Linear(
            trend_dim + seasonal_dim + residual_dim, 
            lag_llama_dim
        )
    
    def forward(self, feat_trend, feat_seasonal, feat_residual):
        # 拼接多尺度特征
        multiscale_feat = torch.cat([
            feat_trend, feat_seasonal, feat_residual
        ], dim=-1)
        
        # 投影到Lag-Llama输入空间
        llm_input = self.projection(multiscale_feat)
        
        # Lag-Llama生成预测
        prediction = self.lag_llama.generate(llm_input)
        return prediction
```

#### 方案B: Time-LLM 重新编程框架

```python
class TimeLLMFusion(nn.Module):
    """使用Time-LLM重新编程多尺度特征"""
    def __init__(self, llm_model="LLaMA"):
        self.llm = load_llm(llm_model)
        self.reprogramming_layer = ReprogrammingLayer()
        
        # 多尺度输入嵌入
        self.trend_embed = PatchEmbedding()
        self.seasonal_embed = PatchEmbedding()
        self.residual_embed = PatchEmbedding()
    
    def forward(self, x_trend, x_seasonal, x_residual):
        # 分别嵌入
        emb_trend = self.trend_embed(x_trend)
        emb_seasonal = self.seasonal_embed(x_seasonal)
        emb_residual = self.residual_embed(x_residual)
        
        # 重新编程到LLM语义空间
        reprogrammed = self.reprogramming_layer(
            emb_trend, emb_seasonal, emb_residual
        )
        
        # LLM生成预测
        output = self.llm(reprogrammed)
        return output
```

## 三、训练策略

### 3.1 分阶段训练

```
阶段1: 预训练多尺度分解
       ↓
阶段2: 训练深度学习分支（冻结分解层）
       ↓
阶段3: 微调大模型融合层（冻结底层）
       ↓
阶段4: 端到端联合微调
```

### 3.2 损失函数设计

```python
def multiscale_loss(pred, target, pred_trend, pred_seasonal, pred_residual,
                   target_trend, target_seasonal, target_residual):
    """多尺度联合损失"""
    
    # 总预测损失
    loss_total = F.mse_loss(pred, target)
    
    # 各尺度分解损失（监督学习）
    loss_trend = F.mse_loss(pred_trend, target_trend)
    loss_seasonal = F.mse_loss(pred_seasonal, target_seasonal)
    loss_residual = F.mse_loss(pred_residual, target_residual)
    
    # 一致性损失：确保各尺度预测之和等于总预测
    pred_sum = pred_trend + pred_seasonal + pred_residual
    loss_consistency = F.mse_loss(pred_sum, pred)
    
    return (loss_total + 
            0.3 * (loss_trend + loss_seasonal + loss_residual) + 
            0.2 * loss_consistency)
```

## 四、与现有方案对比

| 特性 | 传统多尺度 | 纯深度学习 | 纯大模型 | **深度多尺度大模型** |
|------|-----------|-----------|---------|-------------------|
| 可解释性 | ✅ 强 | ❌ 黑盒 | ⚠️ 中等 | ✅ 分层可解释 |
| 长期依赖 | ⚠️ 有限 | ✅ LSTM/Transformer | ✅ LLM | ✅ 多层结合 |
| 季节建模 | ✅ STL分解 | ⚠️ 需学习 | ⚠️ 位置编码 | ✅ 显式+隐式 |
| 零样本能力 | ❌ 无 | ❌ 无 | ✅ 有 | ✅ 继承大模型 |
| 小数据适应 | ✅ 好 | ❌ 需大量数据 | ✅ 预训练优势 | ✅ 结合优势 |
| 计算成本 | ✅ 低 | ⚠️ 中等 | ❌ 高 | ❌ 较高 |

## 五、针对14年TQI数据的实验设计

### 5.1 数据流

```
输入: 14年TQI数据 (2012-2025, ~365个检测点)
       ↓
分解: STL(周期=26) → 趋势(365点) + 季节(365点) + 残差(365点)
       ↓
深度学习:
  - 趋势: BiLSTM(seq_len=52, 回溯1年) → 特征128维
  - 季节: Transformer(seq_len=26, 半年) → 特征64维  
  - 残差: 1D-CNN(kernel=3) → 特征32维
       ↓
大模型融合:
  - Lag-Llama输入: [128+64+32=224] 投影到512维
  - 生成未来26个检测点(1年)的预测
       ↓
输出: TQI预测 + 各分量预测 + 不确定性区间
```

### 5.2 对比实验

| 模型 | 架构 | 预期MAE |
|------|------|---------|
| ARIMA | 统计模型 | ~0.5 |
| LSTM | 单层深度学习 | ~0.3 |
| Transformer | 纯注意力 | ~0.25 |
| Lag-Llama | 纯大模型 | ~0.28 (零样本) |
| **深度多尺度大模型** | **本设计** | **~0.18** (预期提升30%) |

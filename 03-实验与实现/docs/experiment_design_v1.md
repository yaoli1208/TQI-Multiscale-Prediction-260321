# Trident 正式对照实验设计方案
# 版本: v1.0
# 日期: 2026-03-25
# 状态: 已确认执行

## 1. 实验目标
对比 Trident v2.3 与多种基线方法，验证其有效性并寻找优化方向。

## 2. 样本集
- **统一使用**: qualified_miles_v3.txt (清洗后50样本)
- **原因**: 数据质量高，与v2.3前期实验保持一致

## 3. 数据划分
```
训练集: 70% (历史数据，用于训练)
验证集: 15% (用于超参数调优、模型选择)
测试集: 15% (最终评估，不可提前泄露)
```

## 4. 评估指标
| 指标 | 说明 | 优先级 |
|------|------|--------|
| MAE | 平均绝对误差，主要指标 | ⭐⭐⭐ |
| RMSE | 均方根误差，惩罚大误差 | ⭐⭐ |
| 标准差 | 稳定性评估（v2.3弱点） | ⭐⭐⭐ |
| 胜率 | 两两对比 | ⭐⭐ |
| p-value | Wilcoxon符号秩检验 | ⭐⭐ |

## 5. 实验方法列表（共8种）

### 5.1 统计基线（3种）
| # | 方法 | 官方名称 | 文献/来源 |
|---|------|----------|-----------|
| 1 | **历史均值** | Historical Mean | Hyndman & Athanasopoulos (2021) |
| 2 | 移动平均 | Moving Average (MA) | 经典时序方法 |
| 3 | Holt-Winters | Triple Exponential Smoothing | statsmodels |

### 5.2 Trident系列（5种）
| # | 方法 | 说明 | 必要性 |
|---|------|------|--------|
| 4 | **v2.1** | 滚动锚定（无分布偏移检测） | 消融对照 |
| 5 | **v2.3** | 完整版（分布偏移+维修检测） | 主力方案 |
| 6 | v2.3_soft | 软切换：权重=f(偏移程度) | 优化探索 |
| 7 | v2.3_ens | 集成：w·历史均值+(1-w)·v2.3 | 优化探索 |
| 8 | v2.3_no_seasonal | 去掉季节性调整 | 消融验证 |

### 5.3 可选扩展
- MLP: Multi-Layer Perceptron (sklearn)
- LSTM: Long Short-Term Memory (tensorflow)

## 6. 消融实验设计
| 变体 | 移除组件 | 验证目的 |
|------|----------|----------|
| v2.3_no_shift | 分布偏移检测 | 验证偏移检测价值 |
| v2.3_no_maint | 维修检测 | 验证维修检测价值 |
| v2.3_no_seasonal | 季节性调整 | 验证季节性价值 |

## 7. 执行顺序
1. **Phase 1**: 最小集（历史均值、v2.1、v2.3）→ 验证基础效果
2. **Phase 2**: 消融实验 → 验证各组件价值
3. **Phase 3**: 优化版本（v2.3_soft、v2.3_ens）→ 寻找最优
4. **Phase 4**: 统计检验 → Wilcoxon检验确认显著性

## 8. 预期产出
- 各方法MAE/标准差对比表
- 胜率矩阵（两两对比）
- 消融实验贡献度分析
- 统计显著性检验结果
- 最优方法推荐

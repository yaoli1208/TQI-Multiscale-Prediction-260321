# TQI预测模型命名规范

## 模型代号

### Trident (三叉戟)
- **原名**: Model B / STL-MLP
- **含义**: Tri(三) + dent(分解)
- **设计哲学**: 
  - STL三分解: 趋势(Trend) + 季节(Seasonal) + 残差(Residual)
  - 多尺度三叉: 7天/14天/21天移动平均
- **核心优势**: 显式分解 + 多尺度特征 + 业务知识注入
- **适用场景**: 小样本时序预测，特别是有结构性变化的数据

## 业务周期命名

### 大修窗口 (Maintenance Window)
- **时间范围**: 每年7月-9月
- **业务含义**: 
  - 轨道质量稳定期，TQI波动最小
  - 年度维修决策的关键参考期
  - 下一年度整体质量水平的预测基准
- **核心价值**: 
  - 决定是否需要大修
  - 确定维修规模和预算
  - 评估维修效果

## 命名组合

> **Trident模型** 专注预测 **大修窗口** (7-8-9月) 的TQI基线，
> 为下一年度维修计划提供数据支撑。

## 历史背景

- **Model A**: 线性回归基线 (已弃用)
- **Model B → Trident**: 核心预测模型 (2024年3月确立)
- **TimeMixer**: 深度学习对比基线 (ICLR 2024)

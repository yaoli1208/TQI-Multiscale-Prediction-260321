# Trident框架实验报告 - 514个有效样本批量实验

**实验日期**: 2026-03-23  
**数据来源**: iic_tqi_all.xlsx (435,753条原始记录)  
**有效样本**: 514个 (从1,203个筛选样本中识别)  
**实验目标**: 验证Trident框架在大规模真实数据上的适用性和性能

---

## 一、实验概述

### 1.1 背景

Trident框架是一种业务经验增强的多尺度轨道质量指数(TQI)预测方法，通过显式编码大修期锚定值、季节性规律和劣化趋势，实现对TQI的精准预测。

本实验对1,203个轨道样本进行批量测试，识别出514个Trident有效的样本，并深入分析其特征和性能表现。

### 1.2 实验设计

| 项目 | 设置 |
|:---|:---|
| 数据来源 | iic_tqi_all.xlsx |
| 原始记录数 | 435,753条 |
| 筛选后样本数 | 1,203个 (记录数100-2000) |
| 有效样本数 | 514个 |
| 数据时间跨度 | 2012-2026年 |

### 1.3 三种预测策略

| 策略 | 核心逻辑 | 适用场景 |
|:---|:---|:---|
| **数据驱动** | 训练集历史均值作为常数预测 | 基线对比 |
| **滚动锚定** | 最近3年度均值 + 季节性调整 | 有年度周期性规律 |
| **修后预测** | 最近大修后锚定值 + 季节性调整 | 已知维修周期 |

### 1.4 有效性判断标准

Trident有效的定义：滚动锚定或修后预测的MAE相比数据驱动提升 **> 10%**

## 二、核心实验结果

### 2.1 总体统计

| 指标 | 数值 |
|:---|:---:|
| **总样本数** | 1,203个 |
| **Trident有效样本** | **514个 (42.7%)** |
| **Trident无效样本** | 689个 (57.3%) |

### 2.2 性能提升统计

| 指标 | 数值 |
|:---|:---:|
| 平均MAE改善 | **30.5%** |
| 最大MAE改善 | **83.8%** |
| 中位数MAE改善 | **27.2%** |

### 2.3 MAE改善分布

| 改善幅度 | 样本数 | 占比 |
|:---:|:---:|:---:|
| 10%-20% | 170个 | 33.1% |
| 20%-30% | 112个 | 21.8% |
| 30%-40% | 92个 | 17.9% |
| 40%-50% | 76个 | 14.8% |
| 50%-60% | 37个 | 7.2% |
| 60%-70% | 15个 | 2.9% |
| 70%-80% | 9个 | 1.8% |
| 80%-90% | 3个 | 0.6% |

**结论**: 大多数有效样本(72.8%)的MAE改善在10%-40%之间，少量样本(12.5%)改善超过50%，表现优异。

### 2.4 最佳策略分布

| 策略 | 样本数 | 占比 |
|:---|:---:|:---:|
| **滚动锚定** | 306个 | **59.5%** |
| **修后预测** | 208个 | 40.5% |

**结论**: 滚动锚定策略在多数有效样本上表现最优，无需预知大修时间点，更具实用性。

## 三、有效样本特征分析

### 3.1 基本统计特征

| 特征 | 有效样本 | 无效样本 | 差异 | 显著性 |
|:---|:---:|:---:|:---:|:---:|
| 平均记录数 | 368条 | 318条 | +50 | 数据量更充足 |
| 平均时间跨度 | 10.4年 | 9.4年 | +1年 | 历史更长 |
| TQI均值 | 2.74 | 2.50 | +0.24 | TQI水平略高 |
| **TQI标准差** | **0.45** | **0.36** | **+0.10** | **波动更大** |
| **季节性振幅** | **0.56** | **0.46** | **+0.10** | **季节性更强** |

**关键发现**: Trident适用于有一定波动(σ>0.35)和明显季节性(振幅>0.40)的轨道。

### 3.2 样本类型分析

| 样本类型 | 定义 | 数量 | Trident有效率 | 平均改善 |
|:---|:---|:---:|:---:|:---:|
| 稳定型 | TQI<2.5, σ<0.4 | 565个 | 37.9% | 25.8% |
| **波动型** | **TQI 2.5-3.5, σ>0.6** | **151个** | **62.9%** | **37.5%** |
| 高TQI型 | TQI>3.5 | 487个 | 44.6% | 32.1% |

**结论**: 波动型样本最适合Trident，有效率高达62.9%，平均改善37.5%。

## 六、Trident适用性判断规则

### 6.1 推荐阈值

| 指标 | 建议阈值 | 说明 |
|:---|:---:|:---|
| TQI标准差 | > 0.35 | 需要有一定波动 |
| 季节性振幅 | > 0.40 | 需要有年度周期 |
| 时间跨度 | > 5年 | 需要足够历史 |
| 记录数 | > 200 | 需要足够数据量 |

### 6.2 决策流程

```
TQI标准差 > 0.35?
├── 否 → 数据驱动（过于平稳）
└── 是 → 季节性振幅 > 0.40?
    ├── 否 → 数据驱动
    └── 是 → 时间跨度 > 5年?
        ├── 否 → 数据驱动
        └── 是 → Trident框架
            ├── 有清晰大修周期 → 修后预测
            └── 无大修周期 → 滚动锚定
```

## 八、局限性与未来工作

### 8.1 当前局限

1. **57.3%样本Trident无效**：需要数据驱动基线
2. **劣化率固定**：未学习样本特异速率
3. **大修检测依赖阈值**：可能误判或漏检
4. **季节性为固定月度偏差**：未考虑年际变化

### 8.2 改进方向

1. **自适应劣化率**：从历史数据拟合每段轨道的劣化系数
2. **维修记录融合**：接入正式维修计划，精确锚定
3. **多源数据融合**：引入气象、通车量数据
4. **不确定性量化**：提供预测区间而非点估计

## 十、结论

### 10.1 核心发现

1. **Trident框架在大规模数据上表现稳定**
   - 42.7%的样本Trident有效
   - 有效样本平均MAE改善30.5%
   - 最佳样本改善达83.8%

2. **滚动锚定策略最实用**
   - 59.5%有效样本选择该策略
   - 无需预知大修时间点
   - 适用于周期性规律明显的轨道

3. **波动型样本最适合Trident**
   - 有效率高达62.9%
   - 平均改善37.5%
   - 业务经验价值显著

### 10.2 工程价值

- **可部署性**: 42.7%的轨道区段可直接应用
- **效果明确**: 平均30.5%的MAE改善
- **成本可控**: 仅需历史检测数据，无需额外传感器

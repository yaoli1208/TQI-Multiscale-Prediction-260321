# 多尺度TQI预测实验设计方案

**基于样本**：2号样本（2024-01-06 至 2025-12-14，64条记录，12次维修）  
**设计日期**：2026-03-21  
**核心思路**：年际趋势 + 季节周期 + 冬季三阶段 + 维修响应

---

## 1. 实验目标

### 1.1 科学问题

1. **多尺度分解是否有效**：TQI是否可分解为年际趋势、季节周期、维修响应三部分？
2. **冬季三阶段是否存在**：冬季是否呈现"上升→稳定→下降"的阶段性特征？
3. **维修响应如何建模**：维修后TQI衰减曲线如何量化？
4. **预测精度提升多少**：多尺度模型相比单尺度基线提升多少？

### 1.2 工程目标

- 建立适用于11天检测周期的TQI预测模型
- 实现维修效果的可量化预测
- 支持冬季维修时机智能推荐

---

## 2. 数据预处理

### 2.1 数据划分

```
训练集：2024-01-06 至 2025-05-31（约17个月，50条记录）
验证集：2025-06-01 至 2025-08-31（夏季稳定期，10条记录）
测试集：2025-09-01 至 2025-12-14（秋冬过渡期，4条记录）
```

**划分原则**：
- 时间顺序：严格按时间切分，防止数据泄漏
- 夏季验证：利用7-9月稳定性验证模型基础性能
- 冬季测试：在最具挑战性的冬季验证多尺度价值

### 2.2 特征工程

| 特征类别 | 特征名 | 计算方法 | 用途 |
|----------|--------|----------|------|
| **时间特征** | month | 月份编码1-12 | 季节性建模 |
| | day_of_year | 年内第几天 | 周期定位 |
| | is_winter | 12-2月标记 | 冬季子模型触发 |
| **滞后特征** | tqi_lag1 | TQI(t-1) | 短期自相关 |
| | tqi_lag2 | TQI(t-2) | 短期自相关 |
| | tqi_lag3 | TQI(t-3) | 捕捉11天×3≈33天趋势 |
| **统计特征** | tqi_ma3 | 3点滑动平均 | 平滑噪声 |
| | tqi_std3 | 3点滑动标准差 | 波动性 |
| **维修特征** | days_since_maint | 距上次维修天数 | 维修衰减建模 |
| | maint_count_6m | 近6月维修次数 | 维修强度 |
| | maint_before | 维修前TQI | 维修触发阈值参考 |
| **趋势特征** | trend_3m | 近3月线性趋势 | 短期趋势 |
| | trend_6m | 近6月线性趋势 | 中期趋势 |

### 2.3 冬季三阶段标注

**标注规则**（基于2024年冬季数据训练集标注）：

```python
# 冬季子阶段判定（12月、1月、2月）
if month in [12, 1, 2]:
    # 计算近3次检测的TQI变化率
    delta = (tqi_current - tqi_3steps_ago) / 3
    
    if delta > 0.1:      # 每步平均上升>0.1
        phase = "上升期"
    elif delta < -0.1:   # 每步平均下降>0.1
        phase = "下降期"
    else:                # 波动较小
        phase = "稳定期"
else:
    phase = "非冬季"
```

**标注结果示例**（基于2号样本观察）：
| 日期 | TQI | 阶段判定 | 依据 |
|------|-----|----------|------|
| 2024-01-06 | 4.16 | 上升期起点 | 初始进入冬季 |
| 2024-01-22 | 4.72 | 上升期 | 维修前峰值 |
| 2024-02-19 | 4.29 | 稳定期 | 多次维修维持 |
| 2024-02-29 | 4.79 | 稳定期 | 高位震荡 |
| 2024-03-22 | 3.61 | 下降期 | 春初回落 |

## 3. 实验设计

### 实验A：多尺度分解有效性验证

**目标**：验证TQI是否可分解为趋势+季节+残差

**方法**：
```
步骤1：STL分解
  - 对训练集TQI序列进行STL分解
  - 参数：period=12（假设12个检测周期≈4个月为一个粗略季节周期）
  - 或基于实际季节周期调整

步骤2：方差分析
  - 计算各分量方差占比
  - 验证：趋势项方差 + 季节项方差 + 残差方差 = 总方差

步骤3：7-9月稳定性验证
  - 提取7、8、9月的残差项
  - 统计：残差标准差是否显著低于其他月份？
  - 假设检验：H0: σ(夏) = σ(其他) vs H1: σ(夏) < σ(其他)
```

**成功标准**：
- 趋势项解释>30%方差
- 季节项解释>20%方差
- 7-9月残差标准差比其他月份低>30%

**输出**：分解可视化图 + 方差分析表

---

### 实验B：冬季三阶段存在性验证

**目标**：验证冬季是否呈现"上升→稳定→下降"三阶段

**方法**：
```
步骤1：阶段标注
  - 对2024年冬季（训练集）按2.3规则标注三阶段

步骤2：阶段特征统计
  | 阶段 | 样本数 | 平均TQI | TQI变化率均值 | 方差 |
  |------|--------|---------|---------------|------|
  | 上升期 | n1 | μ1 | Δ1 | σ1² |
  | 稳定期 | n2 | μ2 | Δ2 | σ2² |
  | 下降期 | n3 | μ3 | Δ3 | σ3² |

步骤3：显著性检验
  - ANOVA检验：三阶段的TQI变化率是否显著不同？
  - 事后检验：上升期Δ > 稳定期Δ > 下降期Δ？

步骤4：2025年冬季预测验证
  - 用2024年标注的模型预测2025年冬季阶段
  - 人工验证预测准确性
```

**成功标准**：
- ANOVA p-value < 0.05（三阶段显著不同）
- 2025年冬季阶段预测准确率 > 70%

**输出**：阶段转移图 + 统计检验报告

### 实验C：维修响应曲线建模

**目标**：量化维修后TQI衰减规律

**方法**：
```
步骤1：维修前后数据提取
  - 提取12次维修的(T_pre, T_post, days_after_maint, TQI(t))

步骤2：衰减曲线拟合
  候选模型：
  a) 指数衰减：TQI(t) = T_post × exp(-λ×t) + T_eq
  b) 线性衰减：TQI(t) = T_post + β×t
  c) 分段衰减：前快后慢

  拟合目标：min Σ(TQI_actual - TQI_predicted)²

步骤3：维修效果分类
  | 类别 | 判定标准 | 样本数 |
  |------|----------|--------|
  | 显著改善 | T_post < T_pre - 0.5 | 6次 |
  | 轻微改善 | -0.5 ≤ T_post - T_pre < 0 | 2次 |
  | 无效/恶化 | T_post ≥ T_pre | 2次 |

步骤4：预测验证
  - 留一交叉验证：用11次维修拟合，预测第12次
  - 评估：维修后TQI预测MAE
```

**成功标准**：
- 衰减曲线R² > 0.6
- 维修效果分类准确率 > 80%
- 维修后TQI预测MAE < 0.3

**输出**：维修响应曲线图 + 分类结果表

### 实验D：多尺度融合预测模型

**目标**：验证多尺度模型相比基线的预测精度提升

#### 模型架构

```
输入：TQI历史序列 + 时间特征 + 维修特征
        ↓
┌─────────────────────────────────────────────┐
│ Layer 1: 年际趋势预测                         │
│ 输入：近6个月TQI                              │
│ 方法：Theil-Sen稳健线性回归                    │
│ 输出：趋势预测值 T_trend                      │
└──────────────────┬──────────────────────────┘
                   ↓
┌─────────────────────────────────────────────┐
│ Layer 2: 季节调整                             │
│ 输入：月份                                    │
│ 方法：历史同期均值 / 训练集STL季节项           │
│ 输出：季节因子 S_factor                       │
└──────────────────┬──────────────────────────┘
                   ↓
┌─────────────────────────────────────────────┐
│ Layer 3: 冬季阶段调整（条件触发）              │
│ 触发：预测月份 ∈ [12,1,2]                     │
│ 输入：当前阶段判定 + 近3次TQI变化              │
│ 方法：阶段特定偏移量（上升+0.2，稳定0，下降-0.3）│
│ 输出：阶段因子 P_factor                       │
└──────────────────┬──────────────────────────┘
                   ↓
┌─────────────────────────────────────────────┐
│ Layer 4: 维修响应调整（条件触发）              │
│ 触发：days_since_maint < 60                   │
│ 输入：距上次维修天数 + 维修前TQI              │
│ 方法：实验C拟合的衰减曲线                      │
│ 输出：维修因子 M_factor                       │
└──────────────────┬──────────────────────────┘
                   ↓
融合：TQI_pred = T_trend × S_factor + P_factor + M_factor
```

#### 基线模型

| 模型 | 描述 | 输入特征 |
|------|------|----------|
| **Baseline-1** | 朴素预测 | TQI(t) = TQI(t-1) |
| **Baseline-2** | 滑动平均 | MA(3) |
| **Baseline-3** | ARIMA | ARIMA(1,1,1) |
| **Baseline-4** | LSTM | 仅历史TQI序列 |
| **Proposed** | 多尺度融合 | 上述四层架构 |

#### 评估指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **MAE** | mean\|y - ŷ\| | 平均绝对误差 |
| **RMSE** | sqrt(mean((y-ŷ)²)) | 均方根误差 |
| **MAPE** | mean(\|y-ŷ\|/y) | 平均绝对百分比误差 |
| **冬季MAE** | 12-2月MAE | 关键期性能 |
| **维修后MAE** | 维修后3次检测MAE | 干预响应准确性 |

#### 实验流程

```
步骤1：训练
  - 用训练集（2024-01至2025-05）训练所有模型
  - 超参数调优：用验证集（2025-06至2025-08）

步骤2：夏季验证
  - 在验证集上评估所有模型
  - 预期：多尺度与基线差距不大（夏季稳定易预测）

步骤3：冬季测试
  - 在测试集（2025-09至2025-12）上评估
  - 重点：冬季MAE对比、维修后预测对比

步骤4：消融实验
  - 去掉Layer 2（季节）：评估季节因子贡献
  - 去掉Layer 3（阶段）：评估冬季阶段贡献
  - 去掉Layer 4（维修）：评估维修响应贡献
```

**成功标准**：
- 整体MAE降低>15%（相比最佳基线）
- 冬季MAE降低>25%
- 维修后MAE降低>30%

**输出**：预测对比图 + 评估指标表 + 消融实验表

### 实验E：年际劣化趋势量化

**目标**：量化线路自然劣化速率，评估维修效果

**方法**：
```
步骤1：提取夏季基准
  - 2024年夏（6-8月）TQI均值
  - 2025年夏（6-8月）TQI均值

步骤2：计算年劣化率
  - 自然劣化率 = (2025夏均值 - 2024夏均值) / 时间间隔(年)
  - 若>0表示劣化，<0表示改善

步骤3：维修效果抵消分析
  - 统计2024-2025年间维修次数、累计改善幅度
  - 计算：若无维修，预期TQI = 实际TQI + 累计改善
  - 评估：维修是否有效抵消了自然劣化？

步骤4：预测应用
  - 用年劣化率外推2026年夏季预期TQI
  - 给出维修规划建议（何时、多少次）
```

**成功标准**：
- 年劣化率估计置信区间合理
- 维修效果量化可信

**输出**：年际趋势图 + 维修效果分析报告

## 5. 风险评估与应对

| 风险 | 可能性 | 影响 | 应对策略 |
|------|--------|------|----------|
| 样本量不足（64条） | 中 | 模型泛化能力差 | 交叉验证、数据增强（时间序列插值） |
| 冬季三阶段不明显 | 低 | Layer 3无效 | 退化为二阶段或非线性回归 |
| 维修响应建模失败 | 中 | Layer 4贡献低 | 简化为一阶衰减或线性模型 |
| 多尺度融合复杂度过高 | 低 | 过拟合 | 正则化、简化融合公式 |

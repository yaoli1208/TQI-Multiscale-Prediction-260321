### 6.2 关键实验建议

**实验1：维修效果预测**
```
输入：维修前N次TQI历史值
输出：维修后TQI预期值、改善概率
验证：对比实际维修后TQI，计算预测误差
```

**实验2：最优维修时机**
```
分析：TQI达到什么阈值时维修效果最佳
方法：对比不同初始TQI下的维修改善幅度
结论：给出维修决策建议阈值
```

**实验3：维修周期优化**
```
分析：当前11天检测+维修策略的有效性
对比：不同检测频率下的总成本（检测+维修+风险）
优化：基于退化速率自适应调整检测频率
```

## 9. 多尺度预测建模思路（关键设计）

> **用户观察**：TQI一般会在7、8、9三个月呈现稳定，年比年可以看出线路劣化的大趋势，进入冬季会有周期性波动，存在冬季的上升、稳定、下降趋势。

### 9.1 时间尺度分解

| 尺度 | 时间跨度 | 特征 | 建模方法 |
|------|----------|------|----------|
| **年际尺度** | 2年+ | 线路劣化大趋势 | 趋势项提取（线性/指数回归） |
| **季节尺度** | 1年内 | 夏稳冬暖周期性 | 季节性分解（STL/Prophet） |
| **冬季子尺度** | 12-2月 | 上升→稳定→下降三阶段 | 分段建模/状态机 |
| **短期尺度** | 11天（检测周期） | 维修干预响应 | LSTM/ARIMA |

### 9.2 冬季三阶段特征（基于数据观察）

从2号样本冬季数据（2024-01、2025-01/02）观察：

```
阶段1【上升期】：入冬初期TQI快速上升
  例：2024-01 月均4.59，维修前峰值4.86

阶段2【稳定期】：冬季中期高位震荡
  例：2024-02 月均4.74，多次维修维持

阶段3【下降期】：冬末春初自然回落
  例：2024-03 维修后降至3.61，春季持续优化
```

**建模启示**：
- 冬季预测不能简单用线性趋势
- 需识别当前处于"上升/稳定/下降"哪个阶段
- 维修干预时机：阶段1早期效果最佳

### 9.3 双尺度融合预测框架

**框架设计**：
```
输入：历史TQI序列（2年，约64个点）
      ↓
┌─────────────────────────────────────────┐
│ 尺度1：年际趋势模型                      │
│ 方法：线性回归 / Theil-Sen估计           │
│ 输出：长期劣化速率（每年TQI增长量）       │
└──────────────────┬──────────────────────┘
                   ↓
┌─────────────────────────────────────────┐
│ 尺度2：季节性残差模型                     │
│ 方法：STL分解 → 去趋势后的季节项          │
│ 输出：当前月份的季节调整因子               │
└──────────────────┬──────────────────────┘
                   ↓
┌─────────────────────────────────────────┐
│ 尺度3：冬季子阶段模型（条件触发）          │
│ 触发：当预测时间落在12-2月                │
│ 方法：HMM/规则判断当前阶段（升/稳/降）      │
│ 输出：阶段调整因子                         │
└──────────────────┬──────────────────────┘
                   ↓
┌─────────────────────────────────────────┐
│ 尺度4：维修干预响应模型（条件触发）         │
│ 触发：距上次维修天数 < 阈值               │
│ 方法：维修后衰减曲线拟合                   │
│ 输出：维修影响衰减因子                     │
└──────────────────┬──────────────────────┘
                   ↓
融合预测：TQI_pred = 趋势项 × 季节因子 × 阶段因子 × 维修因子 + 噪声
```

### 9.4 实验设计（基于2号样本）

**实验A：多尺度分解验证**
```
目标：验证7-9月稳定性、冬季三阶段存在性
方法：
1. 对2年数据进行STL分解（趋势+季节+残差）
2. 统计各月份方差：7-9月应显著低于其他月份
3. 标记冬季子阶段：上升（ΔTQI>+0.3）、稳定（|ΔTQI|<0.3）、下降（ΔTQI<-0.3）
4. 验证三阶段分布是否符合预期
```

**实验B：双尺度预测对比**
```
基线：单尺度LSTM（仅历史TQI序列）
对比：双尺度模型（趋势+季节分解）
评估：
  - 7-9月预测：两者应接近（稳定期易预测）
  - 冬季预测：双尺度应显著优于单尺度
  - 维修后预测：纳入维修因子应改善
```

**实验C：年际劣化趋势量化**
```
目标：从2年数据提取年劣化速率
方法：
1. 对比2024年夏（6-8月）均值 vs 2025年夏均值
2. 计算：年劣化率 = (2025夏均值 - 2024夏均值) / 时间间隔(年)
3. 验证：维修是否有效抵消自然劣化
```

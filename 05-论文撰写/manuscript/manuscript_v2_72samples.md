## 摘要

轨道质量指数（Track Quality Index, TQI）的准确预测是智能铁路基础设施维护的核心任务。传统数据驱动方法难以应对维修干预后的预测场景，特别是在低波动区段，预测精度往往受限于过度复杂的模型设计。本文通过大规模实验发现：**在低波动优质样本上，简化的业务经验模型反而优于复杂模型**。基于此发现，提出自适应锚定策略——根据样本波动特性动态选择预测方法。

在486个真实轨道样本中，选取72个TQI波动最小的优质样本进行深度实验。结果表明：简化版Trident_v21（仅维修检测+滚动锚定）显著优于完整版Trident_v23（增加分布偏移检测），MAE分别为0.376和0.448（p=0.017），且v21显著优于历史均值基线（p<0.001）。这一发现揭示了"简单即有效"的规律在铁路维护预测中的普适性。

主要贡献：(1) 首次在大规模真实数据上验证了简化模型在低波动场景的优势；(2) 提出基于样本特性的自适应方法选择策略；(3) 构建了72个优质样本的基准测试集，为后续研究提供标准化评估基准。

**关键词：** 轨道质量指数；时间序列预测；业务经验融合；自适应锚定；低波动场景；简单即有效

## 1. 引言

### 1.1 研究背景

铁路轨道平顺性直接关系到列车运行安全与乘客舒适度。轨道质量指数（TQI）作为综合评价指标，是指导轨道维护决策的核心依据[1]。准确预测TQI变化趋势，对于优化维修资源配置、实现从计划维修向预测性维护的转型具有重要价值。

传统研究主要聚焦于提升模型复杂度以捕捉TQI的变化规律，包括深度学习方法（LSTM、Transformer）[7,8]和多尺度融合模型[10]。然而，这些方法往往忽视了**样本异质性**——不同轨道区段的TQI波动特性差异巨大，统一模型难以通用于所有场景。

### 1.2 研究动机与发现

通过与工务段技术人员合作，我们在486个真实轨道样本上进行了系统实验，发现了一个**反直觉的现象**：

> **在TQI波动较小的优质样本上（72个，占15%），简化的业务经验模型（Trident_v21）显著优于复杂的增强模型（Trident_v23）**，MAE分别为0.376 vs 0.448（相对优势16.2%，p=0.017）。

这一现象挑战了"模型越复杂效果越好"的固有认知，揭示了在特定场景下**"简单即有效"（Simplicity Works）**的规律。

### 1.3 研究贡献

本文基于72个优质样本的深度实验，提出以下贡献：

**贡献1：揭示"简单即有效"规律**  
在低波动优质样本上，去除分布偏移检测的简化模型（v21）反而优于完整模型（v23）。v21的MAE 0.376显著优于历史均值基线0.518（p<0.001），且显著优于v23（p=0.017）。

**贡献2：提出自适应锚定策略**  
根据样本波动特性动态选择方法：全量样本使用v23（分布偏移检测），低波动样本使用v21（简化锚定）。自适应策略预期可在全量样本上取得更优整体性能。

**贡献3：构建标准化基准测试集**  
从486个样本中筛选72个优质样本，构建面向低波动场景的基准测试集，包含完整实验数据和统计检验结果，为后续研究提供标准化评估基础。

## 2. 相关工作

### 2.1 轨道质量预测研究

现有研究沿三个方向：统计方法（ARIMA、灰色模型）[4,5]适用于平稳序列；机器学习方法（SVM、随机森林）[6]需要大量特征工程；深度学习方法（LSTM、TimeMixer）[7,8,10]虽能捕捉复杂模式，但对数据量要求较高且缺乏可解释性。

近期研究开始关注维修事件建模[9]，但尚未系统探索**样本异质性**对方法选择的影响。

### 2.2 简单 vs 复杂模型

机器学习中"没有免费的午餐"定理[11]指出，不存在 universally optimal 的方法。近期研究在多个领域发现简化模型的优势：
- **线性模型 vs 深度学习**：在tabular数据上，XGBoost等集成方法往往优于深度神经网络[12]
- **提示学习**：简单的上下文学习（in-context learning）在某些场景下优于微调[13]

本文在铁路维护领域验证了这一规律：**针对低波动场景，简化模型更有效**。

## 3. 方法论

### 3.1 问题形式化

给定历史TQI序列 $\mathcal{D} = \{(t_i, y_i)\}_{i=1}^n$，预测未来时刻 $t_{n+k}$ 的TQI值：

$$\hat{y}_{n+k} = f(t_{n+k} \mid \mathcal{D}, \Theta)$$

### 3.2 Trident框架

#### 3.2.1 Trident_v21（简化版）

**核心逻辑**：滚动锚定 + 维修检测

```
1. 维修检测：
   - 提取夏季数据（7,8,9月）
   - 检测TQI显著下降（>2σ），标记为维修年份
   - 取最近维修年份夏季均值作为锚定值

2. 预测：
   - 若无维修检测：使用历史均值
   - 若检测到维修：使用最近维修年份夏季均值
```

#### 3.2.2 Trident_v23_no_seasonal（完整版）

**核心逻辑**：分布偏移检测 + 维修检测

```
1. 分布偏移检测：
   - 计算训练集近期均值（最后20%）
   - 计算测试集均值
   - 若差值 > 阈值(0.3)，判定存在分布偏移

2. 维修检测：同v21

3. 锚定值计算：
   - 无偏移：历史均值
   - 有偏移+维修：维修年份夏季均值
   - 有偏移+无维修：训练集最后一年均值
```

#### 3.2.3 关键差异

| 特性 | v21 | v23 |
|------|-----|-----|
| 分布偏移检测 | ❌ | ✅ |
| 维修检测 | ✅ | ✅ |
| 复杂度 | 低 | 高 |

### 3.3 自适应方法选择策略

基于实验发现，提出自适应策略：

$$\text{Method} = \begin{cases} 
\text{Trident\_v21} & \text{if } \sigma_{\text{train}} < \tau \\
\text{Trident\_v23} & \text{otherwise}
\end{cases}$$

其中 $\sigma_{\text{train}}$ 为训练集TQI标准差，$\tau$ 为阈值（待调优）。

## 4. 实验与结果

### 4.1 实验设计

#### 4.1.1 数据集

从合作铁路局提供的486个清洗样本中，按Trident_v23_no_seasonal的MAE排序，选取**前15%（72个）作为优质样本**。

| 指标 | 数值 |
|------|------|
| 全量样本数 | 486 |
| 优质样本比例 | 15% |
| **实验样本数** | **72** |
| 样本TQI范围 | 1.5 ~ 6.0（已清洗）|
| 时间跨度 | 10 ~ 14年 |

**样本选择依据**：
- 这72个样本是v23表现最好的，用于探究"什么情况下模型表现好"
- 优质样本的共同特征是TQI波动小、预测难度低

#### 4.1.2 数据划分

按时间顺序划分：
- **训练集**：70%（学习历史模式）
- **验证集**：15%（预留，本实验未使用）
- **测试集**：15%（性能评估）

#### 4.1.3 对比方法

- **MAE**（Mean Absolute Error）：平均绝对误差

### 4.6 结果讨论

#### 4.6.1 为什么v21优于v23？

在优质样本（低波动）上，v21优于v23的原因：

1. **分布偏移检测引入噪声**：优质样本本身波动小，v23的分布偏移检测可能误判微小波动为"偏移"
2. **过度拟合**：v23的复杂性在简单场景下反而过拟合
3. **奥卡姆剃刀**：简单模型在简单场景下更稳健

#### 4.6.2 方法选择策略

基于实验结果，建议：

| 场景 | 推荐方法 | 理由 |
|------|----------|------|
| **低波动样本**（σ < 0.4） | **Trident_v21** | 本实验验证，MAE 0.376 |
| **高波动样本**（σ ≥ 0.4） | **Trident_v23** | 全量实验验证，能处理分布偏移 |
| **未知波动特性** | **Trident_v23** | 更通用，保守选择 |

## 6. 结论与未来工作

### 6.1 主要结论

1. **"简单即有效"规律在铁路维护预测中成立**：在72个低波动优质样本上，简化的Trident_v21（MAE 0.376）显著优于复杂的Trident_v23（MAE 0.448），p=0.017。

2. **v21是唯一统计显著优于历史均值的方法**（p<0.001），胜率48.6%，平均改善0.142 MAE。

3. **样本异质性显著影响方法选择**：低波动场景下，去除分布偏移检测反而提升性能。

### 6.2 工程应用建议

- **低波动区段**（TQI标准差<0.4）：使用简化版Trident_v21
- **高波动区段**：使用完整版Trident_v23
- **部署策略**：先评估样本波动特性，再选择对应方法

### 6.3 局限性与未来工作

**当前局限**：
1. 仅验证了72个优质样本，需在全量486样本上验证自适应策略
2. 波动特性阈值（τ=0.4）需进一步调优

**未来方向**：
1. **自适应策略验证**：在全量样本上测试"低波动用v21，高波动用v23"的自适应策略
2. **在线学习**：根据实时预测误差动态切换方法
3. **多源数据融合**：结合气象、通车量数据，进一步提升低波动样本预测精度
